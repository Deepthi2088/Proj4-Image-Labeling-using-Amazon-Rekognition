<h1>Image Labeling using Amazon Rekognition</h1>
<h3>Project Overview</h3>

<p>This project demonstrates how to automatically detect objects and scenes in images using Amazon Rekognition and Python (Boto3).
Images are stored in an Amazon S3 bucket and analyzed by Amazon Rekognition to identify labels with confidence scores.</p>

This project helps understand how to integrate AWS AI services with Python applications.

<h3>AWS Services Used</h3>

Amazon S3 – Stores input images <br>

Amazon Rekognition – Detects labels in images<br>

AWS CLI – Used to configure AWS credentials<br>

Python (Boto3 SDK) – Interacts with AWS services
